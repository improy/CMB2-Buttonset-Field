# CMB2 Buttonset Field Type
CMB2 Buttonset field type. Ready to integrated with a theme

## Use
`buttonset` as a field type.
```php
array(
		'name'    => __( ' Position', 'cmb2' ),Title
	  'id'      => $prefix . 'buttonset',
	  'type'    => 'buttonset',
	  'options' => array(
	       "" => __("Default", 'cmb2'),
	       "left" => __("Left", 'cmb2'),
	       "center" => __("Center", 'cmb2'),
	       "right" => __("Right", 'cmb2')	        
	  ),
	  'default' => 'none',
)  
```
